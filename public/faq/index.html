Frequently asked questions

What IS Code Club World?

Code Club World is a worldwide network of free volunteer-led after-school coding clubs for children aged 9-11. It is based on Code Club UK but the main difference is that volunteer support for Code Club World is offered by the volunteer community themselves.

Creating a community in your country

How do I get involved in Code Club in my country?

Code Club World is community-led. Where we provide the basic projects and materials, we want you to run the community in your own country. Search online, on the Code Club World website, on Twitter or Facebook, and see whether there already is a community in your country.

How do I set up a community in my country?

Use social media to help you find other people who are interested in doing this. Set up a Twitter account for Code Club World in your country.Twitter is great for getting in touch with each other and spreading the word. Maybe use a hashtag for example #CodeClubNorway so that people can find you. Set up a Facebook page for Code Club World in your country, this will help people find each other and it a great place to offer support.

Use forums to hold conversations. We've had great success using free sites such as Google Groups. It’s a great way to support each other, ask questions and generally stay in touch with other volunteers.

Speak to the press to spread the word about your good work. Can you tell your national and local press about Code Club? Maybe they would like to visit some of your clubs?

Arrange meet-ups for local volunteers. In the UK, many of our volunteers organise regular meet ups, where they can meet other volunteers face-to-face, share experiences, updates and news. It also helps to create a sense of community, so you’re not just working on your own.

Setting up a Code Club

How do I set up a Code Club?

If you want to volunteer to run a Code Club, you’ll need to find somewhere suitable to run one. You’ll need to find somewhere with adequate computer facilities with access to the internet.

If you have a suitable venue to run a club, but don’t feel qualified to teach yourself, then you’ll need to find a suitable volunteer!

See below for more details on how to do both of these.

I have a suitable venue but I can’t teach. What can I do?

If you have a venue suitable for running a Code Club - a school, a library, a youth centre, for example - but you don't feel qualified to run a club yourself then you'll need to find a volunteer.

Chat to friends. Email or write home to parents or to local newspapers and community websites. Use Twitter, Facebook, Forums or anything else you can think of.

You'll need to meet with your volunteer and talk about how to take it forward. Agree on a day of the week and a time.

I can teach but I don’t have a place to run a club. What can I do?

If you want to volunteer to run a Code Club, you’ll need to find somewhere suitable to run one. You’ll need to find somewhere with adequate computer facilities with access to the internet.

Primary schools in the UK have been highly successful as students can learn with their friends and grow and develop together. Libraries and community centres, too, might be a good place to run one outside of school hours and weekends.

The important thing is to contact the venue directly, and organise to meet them face-to-face. Explain what Code Club is and take along any relevant materials. Then you can propose a regular day and time to set up your club! Then register your club on this website and download the materials from Github.

The materials aren’t available in my language! What should I do?

Code Club World is community-led, therefore we rely on the good-nature of our volunteers to help translate the materials. If you think you can translate it yourself, or know someone who is able and willing to, then great! The more languages our materials are available in the better.

If you can’t translate yourself, or can’t find anyone who can, why not put a call out on twitter or any forums.

What software do I need?

We start with Scratch. At present, our projects are designed to run on Scratch 1.4 as Scratch 2.0 requires an internet connection and uses Flash, and we want to make Code Club available to the most people possible! We’ll update the projects when the downloadable version of Scratch 2.0 is released later in 2013.

Are there legal requirements?

In the UK any adult who wishes to work with children must have a background criminal record check conducted. This varies from country to country, so you should check with your own country’s laws and make sure you comply with them.

You may also need to have insurance.

How much does it cost?

Code Club is and always will be free! It’s one of our core beliefs and we aim to keep it this way. Code Clubs should be totally volunteer-led, so please don’t charge for your clubs!

I need to find funding to run Code Club in my country - where can i find it?

Look to local technology companies or other local businesses for help as well as grant giving organisations in your region. You might even want to try asking your government for some money!

Contacting Code Club

Where can I get support?

We’re a very small operation here in the UK and don’t have the resources to offer support outside this small island. If you need support look for your country’s Code Club community. Is there already a Code Club website in your country? Look for your country’s community on Twitter or on Google Groups. It might be that there are people out there who can help you.

Can I donate to Code Club World?

Yes! We rely heavily on donations to continue growing and being able to teach as many children to code. You can donate to us online here

Are you a charity?

We are a not-for-profit organisation. This means that although we are not a charity, any money that Code Club makes goes straight back into our work and operations.

Where are you based?

We are a small team based in London, UK. Although our network of wonderful volunteers are based all over the country, and now - the world!

Can my company sponsor Code Club World?

Absolutely! If you are a company interested in sponsoring us, please get in touch via hello@codeclubworld.org

I'm from the press, can I talk to someone at Code Club World.org?

Of course! Please email us at hello@codeclubworld.org to say hello.
