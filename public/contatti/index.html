Contattaci e Supportaci!

Il supporto principale di cui ha bisogno Code Club Italia e' una rete capillare di Volontari e di Aule. Se vuoi supportare le attivita' di Code Club Italia diventa un volontario ed insegna la programmazione ai ragazzi della tua citta'. Oppure metti a disposizione un'aula, un logo adatto alle nostre attivita'. Oppure fai entrambe le cose. Aiutaci a divulgare le attivita' di Code Club Italia, coinvolgi i tuoi amici, conoscenti e familiari.

Se hai qualche dubbio su come diventare un volontario di Code Club Italia, o su come registrare il tuo Club oppure se hai domande sui diversi progetti contattaci via email, con un messagio privato di Twitter o Facebook oppure usa l'apposito gruppo google dedicato a Code Club Italia.
